## Steps to get started

##### Get the app

##### Open the app folder

##### Make sure you have npm/yarn installed.

To Check your version of NPM, use the command:

```
$ npm -v
```
OR

To Check your version of Yarn, use the command:

```
$ yarn -v
```

##### Install dependencies

```
$ npm install
```

OR

```
$ yarn install
```

##### Run the app

```
$ npm start
```

OR

```
$ npm start
```

#### Happy hacking
